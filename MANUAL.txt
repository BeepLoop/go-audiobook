For changing voice setting:

    - Open voice_config.json inside config folder.
    - Change the id to id of new voice.
    - Change the voice_engine to the voice_engine of the chosen voice id.

    - You can view the available voice configurations in the voice_cheatsheet.json inside config folder.

    - Most voices have samples. You can listen to samples of the voice by pasting the url of the sample to you browser.


NOTE:
    - Make sure voice id and voice engine is exactly the same as what is listed in the voice_cheatsheet.json.

    - Never change or modify values of  playht_user and api_key. You only need to change this if you want to use different PlayHT account.
        You can refer to PlayHT website for creating your personal account, https://play.ht/
